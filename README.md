# Vitrine de Ofertas

Pequena aplicação desenvolvida em Next.js que simula uma vitrine de produtos.

## Tecnologias

- Next.js
- React
- Sass
- BEM CSS methodology

## Funcionalidades

- Página `/ofertas`
- Banner promocional
- Vitrine com 6 produtos
- Consumo de API pública (Fake Store API)
- Skeleton loading
- Layout responsivo
- Hover interactions nos produtos

## Como rodar o projeto

```bash
npm install
npm run dev
